# Trading Dashboard - Real-World Use Cases

## Use Case 1: Active Station Trader

**Profile:** Sarah trades in Jita 4-4, focusing on high-volume margin trading

**Morning Routine:**
1. Opens EVE Trade dashboard at `/dashboard`
2. Sets filters:
   - Min Profit: 500,000 ISK (her minimum per trade)
   - Max Investment: 50,000,000 ISK (available liquid ISK)
3. Reviews Top Opportunities → Station Trades
4. Spots "Large Shield Extender II" with 6.25% margin
5. Clicks "Copy Name" to copy to clipboard
6. Pastes in EVE Online market search
7. Clicks "Watch" to add to watchlist for monitoring
8. Uses Quick Calculator to verify:
   - Buy: 3,200,000 ISK
   - Sell: 3,400,000 ISK
   - Quantity: 10
   - Net Profit: 1,840,000 ISK (after fees)
   - ROI: 5.75%
9. Executes trade in-game
10. Returns to dashboard, clicks "+10M" (made 1.84M, rounded up)
11. Repeats process

**End of Day:**
- Session Stats shows: 150M ISK earned, 23 trades, 3h 15m trading
- ISK/hour: 46.15M ISK/hr
- Compares to her goal of 40M ISK/hr - successful session!
- Clicks "Reset" to start fresh tomorrow

## Common Workflows Across All Users

### Quick Profit Check
1. See opportunity in opportunities panel
2. Copy to calculator
3. Verify net profit after fees
4. Execute if meets threshold

### Watchlist Management
1. Find good opportunity
2. Click "Watch"
3. Monitor in Watchlist Quick View
4. Copy name when ready to trade
5. Remove when no longer profitable

### Session Tracking
1. Start trading session
2. Use quick add buttons after each trade
3. Monitor ISK/hour in real-time
4. Adjust strategy if below goal
5. Reset at end of session

### Alert Response
1. Alert triggers in Active Alerts panel
2. Check current value
3. Use calculator to verify profit
4. Execute trade if worthwhile
5. Dismiss alert
