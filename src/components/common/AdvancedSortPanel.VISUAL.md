## Summary

The AdvancedSortPanel provides a clean, intuitive interface for multi-column sorting with:

- **Collapsed state** showing summary info
- **Expanded state** revealing all controls
- **6 preset strategies** for quick sorting
- **Custom multi-column** sorting with priority
- **Visual indicators** for all states
- **Responsive design** for mobile and desktop
- **EVE Online theme** with cyan/gold accents
- **Smooth animations** for all interactions

All visual elements follow the EVETrade design system with space-themed backgrounds and accent colors.
